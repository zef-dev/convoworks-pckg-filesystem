Media Context

Setup params for MP3 files.

When to use?

When we want to create media player with mp3 files and play it through device.

How to use:

  1. In the CONTEXT ID field, enter the same name as the name of the Media block, example: filesystemMediaContext
  2. In the MP3 FOLDER PATH field, enter the path to folder with mp3 files, example: C:/Users/Admin/Documents/mp3
  3. In the PUBLIC URL field, enter the public URL to expose songs from, example: https://www.example.com/mp3/
